HTML and CSS exercises for Workshop.

Summary

Learn how to create basic HTML and CSS website by organizing styling images and text.

Description

HTML and CSS are the two primary languages you need to know to build a website. Although there are more
advanced ways to build a site all web developers must understand HTML and CSS regardless of other
technologies they use.

The course starts by exploring HTML tags and how to organize different types of information on HTML pages, HTML
or Hypertext Markup language describes a heirarchy of visual elements that a browser can display. You'll learn about
the most common elements, like headers, paragraphs, tables and lists.

As you pick up HTML, the course will teach you CSS or Cascading Style Sheets. While HTML describes the meaning
and hierarchy of our content, it doesn't describe the presentation of HTML content. CSS allows you to combine vitality
and glamour to make your web site beautiful and engaging.

By the end of this course, you'll be able to combine HTML and CSS to make a web site.

Objectives

After this course, students should be able to:
  • Format text content and links with HTML.
  • Change how content is displayed with CSS.
  • Reuse CSS across multiple pages and arrange content in well defined sections.
  • Add content and background images to a site.
  • Customize fonts and typography.

Each exercise includes its own specific objectives.
